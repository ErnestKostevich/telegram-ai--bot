**Сильные стороны:** streaming как у ChatGPT, BYOK, голос, персоны, мини-игры, реальная память диалога, безопасные группы. Большинство фич уровня production.

**Слабые стороны:** высокий порог входа (нужен API-ключ), нет органического роста, JSON-storage упрётся в потолок на ~500 активных юзерах, нет тестов и CI, монетизация ручная.

---

## 🎯 Стратегия

Маленький проект с потенциалом. Тратить время надо на то, что **движет метрики**, не на инфраструктурную полировку «на вырост». Три цикла:

1. **Уменьшить friction первого использования** → больше людей доходят до wow-момента
2. **Дать поводы возвращаться** → ежедневные/недельные хуки
3. **Дать поводы рекомендовать** → виральные петли

Параллельно — точечно закрывать технический долг там, где он реально мешает.

## 🚀 Phase 1 — Friction-killer (1–2 недели, MUST)

Цель: пользователь должен получить wow-момент **за 30 секунд**, без копания в API-ключах.

### 1.1 🔥 Free Tier через shared-key (highest impact)
**Что:** добавить вариант «попробовать без ключа» — несколько бесплатных AI-запросов в день на общем боте через ключ создателя.
- Лимит: 10 сообщений/день для не-VIP без своего ключа
- Использовать самый дешёвый провайдер (Groq Llama 8B бесплатный, или DeepSeek)
- Счётчик в `user.free_tier_used_today` со сбросом по UTC-дню
- В welcome: «Уже работает! Напиши что угодно. Свой ключ нужен для тяжёлых моделей»

**Effort:** 1 день
**Impact:** ⭐⭐⭐⭐⭐ — главный барьер входа

### 1.2 🎬 Onboarding-wizard
**Что:** при первом `/start` — 3-шаговый интерактивный мастер с inline-кнопками:
- Шаг 1: «Выбери язык»
- Шаг 2: «Хочешь использовать бесплатный режим или свой ключ?»
- Шаг 3: Если ключ → 2-step пикер; если free → «Готово, попробуй!»

**Effort:** 1 день
**Impact:** ⭐⭐⭐⭐

### 1.3 🎁 Inline mode
**Что:** `@AI_DISCO_BOT [вопрос]` в любом чате → AI отвечает inline-result.
- Telegram inline `chosen_inline_result`
- Без ключа → free tier; с ключом → свой провайдер
- Виральная фича — бот работает в чужих чатах

**Effort:** 2 дня (Telegram inline mode + кэширование)
**Impact:** ⭐⭐⭐⭐⭐ — органический рост

### 1.4 ⚡ Action buttons под AI ответом
**Что:** под каждым ответом 3 кнопки: 🔄 Перегенерировать · 📋 Скопировать · 💾 Сохранить как заметку
- Кнопка регенерации → новый ответ заменяет старый
- «Копировать» → отправляет тот же текст без форматирования
- «В заметку» → создаёт `/note` автоматом

**Effort:** 1 день
**Impact:** ⭐⭐⭐ — UX-полишинг, удержание

### 1.5 🌐 `/share` — deep-link приглашение
**Что:** `/share` → готовое сообщение для пересылки с реферальной меткой `?start=ref_USERID`
- Тот, кто пришёл по ссылке → пригласившему +1 в `user["referrals"]`
- Топ-5 рефереров получают временный VIP

**Effort:** 1 день
**Impact:** ⭐⭐⭐⭐

## 🔁 Phase 2 — Retention (2–4 недели, SHOULD)

Цель: дать поводы возвращаться **каждый день**.

### 2.1 📅 Морнинг-дайджест
**Что:** опт-ин на ежедневное утреннее сообщение (8:00 локального времени):
- Прогноз дня от AI (как `/today`)
- Список незавершённых задач из `/todo`
- Активные напоминания
- «Ваш streak: N дней»

**Effort:** 2 дня (нужен per-user timezone)
**Impact:** ⭐⭐⭐⭐

### 2.2 🧩 Многоходовой квиз
**Что:** `/quizgame [тема] [N]` — серия из N вопросов с финальным результатом и местом в weekly leaderboard.
- Сохраняется в `user["quiz_stats"]["sessions"]`
- Кнопка «следующий вопрос» вместо отдельных команд

**Effort:** 2 дня
**Impact:** ⭐⭐⭐

### 2.3 🎙 Голосовой **ответ** (TTS)
**Что:** опция «отвечать голосом» — после AI-ответа бот синтезирует через OpenAI TTS и присылает voice-message.
- `/voice on|off` — включить/выключить
- Только в личке
- Цена: TTS дороже чем чат, лимит длины ответа

**Effort:** 1–2 дня
**Impact:** ⭐⭐⭐ — wow-фактор продолжения streaming

### 2.4 🔔 Smart-напоминания
**Что:** `/remind завтра в 9 утра позвонить маме` — natural language через AI парсер.
- Скармливать строку модели с запросом распарсить в `{datetime, text}`
- Fallback на текущий формат `/remind 30 текст`

**Effort:** 1 день
**Impact:** ⭐⭐⭐

### 2.5 🏆 Weekly leaderboard + reset
**Что:** еженедельно (воскресенье 23:59 UTC):
- Снапшот топ-10 публикуется в личку всем активным
- Счётчик XP за неделю сбрасывается, abs-XP остаётся
- Топ-3 недели получают значок в профиле

**Effort:** 1 день
**Impact:** ⭐⭐⭐

### 2.6 💾 «AI помнит обо мне»
**Что:** проактивная память. Бот сам предлагает запомнить факты из диалога:
- После N-го упоминания одной темы (имя кота, работа, хобби) → кнопка «🧠 Запомнить»
- Контекст автоматически шире, чем сейчас

**Effort:** 3 дня (требует анализа диалога)
**Impact:** ⭐⭐⭐⭐ — главная фишка против ChatGPT

## 🚀 Phase 3 — Differentiation (1–2 месяца, COULD)

Цель: то, чего нет у других Telegram-AI-ботов. Конкурентный ров.

### 3.1 🌐 Web search в стриме
**Что:** AI может вызвать инструмент `web_search` внутри ответа.
- Использовать Brave/Tavily/Serper API
- В стриминге показывать: «🔍 Ищу: «X»…», потом продолжать ответ
- Tool-calling API провайдеров (OpenAI, Anthropic, Gemini)

**Effort:** 4 дня (tool-calling per provider)
**Impact:** ⭐⭐⭐⭐⭐ — даёт «свежий» AI

### 3.2 🖼 Image-to-image (Vision + Generate)
**Что:** прислал фото с подписью «сделай в стиле Pixar» → результат
- FLUX/SDXL img2img endpoint
- Только VIP

**Effort:** 2 дня
**Impact:** ⭐⭐⭐⭐

### 3.3 💻 Code sandbox
**Что:** `/run python код` — выполнить Python в изолированном sandbox.
- Использовать Piston API (бесплатно) или e2b.dev
- Лимит времени и памяти
- Только VIP

**Effort:** 3 дня
**Impact:** ⭐⭐⭐ — нишево, но фанаты обожают

### 3.4 📊 Telegram Mini App «Dashboard»
**Что:** native Telegram web-app для:
- Просмотра/редактирования заметок и memory
- История диалогов с поиском
- Графики активности
- Управление API-ключами

**Effort:** 1–2 недели (Telegram Web App + лёгкий frontend)
**Impact:** ⭐⭐⭐⭐ — выглядит **очень** круто, оправдывает VIP

### 3.5 🧑‍🤝‍🧑 Group AI с памятью
**Что:** в группе бот ведёт **общую** память на чат (отдельно от personal).
- Memory привязана к chat_id
- Только админы могут модифицировать
- Полезно для рабочих чатов

**Effort:** 2 дня
**Impact:** ⭐⭐⭐

## 🏗 Phase 5 — Scale (когда >500 активных)

Не делать раньше времени.

### 5.1 🗄 Postgres (Render Persistent Disk)
**Что:** перейти с JSON на SQLite (потом Postgres). Сейчас JSON упрётся в:
- ~100KB на пользователя при активном диалоге
- GitHub API rate limit (5000 req/час)
- Время save (1–2 сек на каждый коммит)

**Trigger:** когда `bot_data.json` > 5MB или save() > 3 сек

**Effort:** 4 дня (миграция + dual-write период)

### 5.2 🧪 Тесты и CI
**Что:**
- `pytest` для unit + smoke import
- GitHub Actions: на каждый PR — `py_compile`, `pytest`, `import main`
- Покрытие критичных мест: storage save/load, AI handler, /todo index validation

**Trigger:** при первом баге в проде, который пропустил CI

**Effort:** 2 дня

### 5.3 📊 Логирование и мониторинг
**Что:**
- structured logging с user_id в каждой строке
- Healthcheck endpoint на :8080
- UptimeRobot пингует endpoint
- Sentry для exceptions

**Trigger:** падение бота в проде > 5 минут незамеченно

**Effort:** 1 день

### 5.4 🔁 Multiple Render workers
**Что:** sharded по user_id, или master+worker для тяжёлых задач (vision, generate).
**Trigger:** превышение лимитов одного worker'а
**Effort:** 3 дня

## 📐 Принципы при работе над roadmap

1. **Ship мелкие куски часто.** Лучше 5 пушей по 100 строк, чем один на 500.
2. **Сначала измерь, потом оптимизируй.** Не делать масштабирование пока пользователей нет.
3. **Не повторять ChatGPT.** Делать то, что специфично Telegram'у (inline mode, dice games, voice).
4. **i18n с первого дня.** Каждая новая фича — сразу 3 языка.
5. **Без новых багов.** После каждой фичи — пройтись ultra-review хотя бы по затронутым местам.
